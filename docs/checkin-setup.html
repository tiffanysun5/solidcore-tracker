Solidcore Tracker

Auto Check-in

Checks you in automatically when your phone arrives near the studio (~4 blocks).

⚠️ The check-in mutation is not yet captured. Once you tap "Check in" in the Wellhub app with mitmproxy running, share the network request and this will be fully wired up. The iOS Shortcut below is ready to go.

How it works

1
Your phone detects you're within ~400 m of the studio (4 Manhattan blocks).
2
iOS Shortcuts fires a background HTTP request to GitHub Actions.
3
GitHub runs checkin.py — it finds today's Solidcore booking and calls the Wellhub check-in API.

📍 iOS Shortcut Setup

Do this once per studio location. Takes about 3 minutes.

1
Open the Shortcuts app → tap Automation tab → + → Personal Automation.
2
Choose Arrive → tap Choose → search for 155 W 23rd St (Chelsea) or 37 W 8th St (Greenwich Village). Set radius to 400 meters. Tap Next.
3
Tap Add Action → search Get Contents of URL → tap it.
4
Set the URL to:
https://api.github.com/repos/tiffanysun5/solidcore-tracker/actions/workflows/checkin.yml/dispatches
Method: POST
Headers: add Authorization = Bearer YOUR_GITHUB_TOKEN and Accept = application/vnd.github.v3+json
Body: JSON → {"ref": "main", "inputs": {"source": "ios-shortcut"}}
5
Tap Next → turn off "Ask Before Running" → tap Done.
6
Repeat for the second studio if needed.
